# 亡灵骑兵 / Undead Cavalry
此数据包为Minecraft添加多种骑乘单位（骑兵），结合原版生物生成概率、群系及行为特性。
生成概率参考被骑乘者原版生成条件，并根据骑兵稀有度调整，同时考虑夜间、玩家状态等因素。

## 1.21.10-
---

### 1. 小僵尸骑兵 (Mini Zombie Cavalry)
- **生成群系**：与被骑乘者相同（狼/狐狸原版生物群系）
- **生成几率**：
  - 基础约15%，夜间翻倍至30%
  - 依赖原版狼/狐狸生成方块及群体数量
- **骑乘者**：小僵尸、小尸壳
  - **主手武器**：铁剑
    - **特殊效果**：有概率附魔，魔咒为火焰附加（II）、击退（II）、锋利（V）、耐久（III）、突进（III）（括号内为最大等级，实际为1-max的随机，组合也是随机组合）
  - **装备**：铁质盔甲
    - **特殊效果**：有概率附魔，魔咒为保护（IV）、荆棘（III）耐久（III），随机挑选与组合
  - **特殊效果**：有概率获得药水效果，效果为隐身、迅捷、伤害吸收、力量、抗性提升、蓄风，随机挑选与组合
- **被骑乘者**：狼、狐狸

### 5. 幻翼特攻队 (Phantom Strike Squad)
- **生成群系**：与原版相同
- **生成几率**：与原版相同，基于玩家未睡眠时间增长
- **行为**：
  - 每次攻击且命中时，有概率自爆（自爆后该幻翼死亡）
    - 普通夜晚：苦力怕爆炸效果
    - 雷雨天：闪电苦力怕爆炸效果
  - 自爆时生成两个新的幻翼（新的幻翼依旧可以自爆）

## 备注
1. 所有骑兵单位生成概率参考被骑乘者原版条件，再乘以骑兵稀有系数（约0.3~0.5）
2. 夜间、特定生物群系、玩家状态可对生成概率进行加成或限制
3. 副手盾牌使用仅在远程攻击时触发，冲锋时不使用，增加战术感
4. 数据包目标：保留Minecraft原版生物特色，同时增强骑兵多样性和策略性
